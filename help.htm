Описание программы

Программа Blind_Log предназначена для ведения журнала радиосвязей (QSO). Она позволяет добавлять, редактировать, удалять и экспортировать записи QSO в формате ADIF.

Интерфейс программы

Когда программа открывается, фокус попадает на вкладку "Добавить QSO". В этой вкладке можно ввести данные о радиосвязи, такие как позывной, имя, город, QTH, частота, RST принято, RST передано, режим, диапазон и комментарий.

Переходя по вкладкам с помощью клавиш Ctrl+Tab и Ctrl+Shift+Tab, фокус перемещается между вкладками "Добавить QSO" и "Журнал".

Вкладка "Добавить QSO":

  • Фокус по умолчанию находится на поле "Позывной".
  • Используя клавишу Tab, можно перемещаться между полями ввода: "Позывной", "Имя", "Город", "QTH", "Частота", "RST принято", "RST передано", "режим", "диапазон", "Комментарий", "Дата", "Время" и кнопка "добавить QSO".
  • Клавиша Shift+Tab позволяет перемещаться в обратном порядке.

Вкладка "Журнал":

  • Фокус по умолчанию находится на списке записей QSO.
  • Используя клавишу Tab, можно перемещаться между этим списком и кнопками "Редактировать", "Удалить" и "Экспорт в ADIF".
  • Клавиша Shift+Tab позволяет перемещаться в обратном порядке.

Работа с датой и временем

Программа автоматически устанавливает текущую дату и время в полях "Дата" и "Время" при добавлении QSO. Часовой пояс можно настроить в разделе "Настройки".

Если выбран UTC, время будет отображаться в формате UTC. Если задан пользовательский часовой пояс, время будет скорректировано в соответствии с указанным смещением.

Для изменения даты и времени вручную используйте соответствующие элементы управления в разделе "Добавить QSO".

Поиск позывного на QRZ.ru

Автоматическая подстановка имени и города

Если в настройках включён флажок "Использовать QRZ.ru для подстановки имени и города", программа при вводе позывного и нажатии Enter автоматически подставит имя и город из базы QRZ.ru (если данные найдены).

Внимание!

Для работы поиска нужен не обычный логин/пароль от личного кабинета QRZ.ru, а отдельные данные для XML API.

Получить их можно так:

  1. Зарегистрируйтесь или войдите на QRZ.ru.
  2. Перейдите в раздел управления XML API.
  3. Сгенерируйте или запросите логин и пароль для XML API.
  4. Введите эти данные в настройках программы.

Если флажок не установлен, поиск и соединение с QRZ.ru не выполняются.

Как пользоваться поиском на QRZ.ru

  1. В настройках отметьте "Использовать QRZ.ru для подстановки имени и города".
  2. Введите логин и пароль для XML API QRZ.ru (см. инструкцию выше).
  3. В поле "Позывной" введите нужный позывной и нажмите Enter — имя и город подставятся автоматически, если найдены в базе.

Обновление программы

Для проверки обновлений выберите пункт меню "Справка" → "Проверить обновления" или нажмите горячую клавишу Ctrl+U.

Если доступна новая версия, программа предложит загрузить и установить обновление. Обратите внимание, что перед обновлением необходимо сохранить все несохранённые данные.

После завершения обновления программа автоматически перезагрузится и откроется уже новая версия.

Проверка обновлений при запуске

В настройках появился флажок "Проверять обновления при запуске программы".

Если он отмечен, при каждом запуске Blind_Log автоматически проверяет наличие новой версии.

Если не отмечен — обновление можно проверить вручную через меню "Справка" → "Проверить обновления".

Как включить автоматическую проверку обновлений

  1. Откройте настройки.
  2. Отметьте "Проверять обновления при запуске программы".
  3. При каждом запуске Blind_Log будет проверять наличие новой версии.

Горячие клавиши

  • Ctrl+P - Открыть настройки
  • Ctrl+Q - Выйти из программы
  • Ctrl+Enter - Добавить QSO
  • Ctrl+E - Редактировать выбранное QSO
  • Ctrl+S - Экспортировать QSO в ADIF
  • Delete - Удалить выбранное QSO
  • Shift+F1 - О программе
  • F1 - Справка
  • Ctrl+Tab - Переключение между вкладками вперёд
  • Ctrl+Shift+Tab - Переключение между вкладками назад
  • Ctrl+U - Проверить обновления
  • Enter - Подставить имя и город по позывному из QRZ.ru (если включена настройка)
